Алгоритм сериализации заключается в следующем.

Поскольку порядок элементов в массиве не имеет значения, мы его сортируем по возрастанию.
Повторяющиеся элементы группируем и в выходной строке указываем количество повторных элементов в виде суффикса "*N".
В выходной строке в явном виде записываем только первый элемент, для остальных указываем дельту, показывающую на сколько элемент отличается от предыдущего.
Ради ещё большей экономии вторую и последующие повторяющиеся подряд дельты совсем опускаем.
